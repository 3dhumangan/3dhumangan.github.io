Abstract

Data-driven human image synthesis is a rising topic with a wide variety of industrial demands and applications. We propose the first 3D-aware generative adversarial network (GAN) that synthesizes full-body human images with consistent appearance under different view-angles and body-poses. To tackle the representational and computational challenges in synthesizing the articulated structure of human bodies, we propose a novel generator architecture in which a 2D convolutional backbone is modulated by two mapping networks, namely a 3D pose mapping network and an appearance mapping network. The 3D pose mapping network is formulated as a renderable implicit function which is inherently view-consistent. To preserve this consistency during the 2D synthesis process, we propose a network design with two key aspects: 1) our backbone is a pixel-wise independent convolutional network which avoids positional reference and promotes equalvariance; 2) modulation from the pose mapping network is passed into the backbone by means of spatial adaptive batch normalization instead of instance normalization so that underlying structure is not impaired. The network is adversarially learned from a large collection of web images needless of manual annotation.
